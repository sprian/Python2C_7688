# 給夥伴的簡單測試說明

## 1. 獲取代碼
```bash
git clone https://github.com/sprian/Python2C_7688.git
cd Python2C_7688
```

## 2. 快速測試（3步搞定）
```bash
# 第1步：進入目錄
cd package/azureiotd

# 第2步：編譯測試程序
./scripts/build_test_standalone.sh

# 第3步：部署到設備
./scripts/deploy_enhanced.sh 192.168.1.1 /tmp
```
